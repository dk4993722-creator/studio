# **App Name**: Aapka Pay Wallet

## Core Features:

- Signup/Login: Allow new users to create accounts and existing users to log in.
- User Dashboard: Display user information, balance, and transaction history.
- Add Money: Enable users to add funds to their wallet using various payment methods.
- Team Information: Provide contact details or information about the Aapka Pay team.
- Share App: Allow users to easily share the app with others via a referral link or other methods.
- Complaint Submission: Provide a form for users to submit complaints or issues.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and reliability in financial transactions.
- Background color: A light, desaturated blue (#E5F6FD) to maintain a clean and modern look without distracting from the content.
- Accent color: A complementary orange (#F2994A) used for call-to-action buttons and important notifications to draw attention.
- Body and headline font: 'PT Sans', a humanist sans-serif for clear and accessible text throughout the app.
- Note: currently only Google Fonts are supported.
- Use clean and modern icons to represent different functions within the app. The icons should be consistent in style and easily recognizable.
- Maintain a clean and intuitive layout with a clear hierarchy. Use white space effectively to prevent the dashboard from feeling cluttered.
- Incorporate subtle animations for transactions and feedback, such as a checkmark appearing after a successful money transfer. The animations should be smooth and fast to maintain a professional and polished user experience.
